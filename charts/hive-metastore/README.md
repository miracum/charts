# hive-metastore

![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square)

A Helm chart for deploying the Hive Metastore

**Homepage:** <https://hive.apache.org/>

## Installation

```sh
helm install -n hive-metastore --create-namespace hive-metastore oci://ghcr.io/miracum/charts/hive-metastore
```
